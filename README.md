# 📊 KPI-Drive (Flutter)

Доска задач, построенная на Flutter

---

## 🚀 Функциональность

- 📌 Отображение задач в виде KPI-DRIVE
- 📂 Колонки = `parent_id` (папки/группы)
- 🧩 Drag & Drop задач между колонками
- ↕️ Изменение порядка задач внутри колонки (`order`)
- 🔍 Поиск задач по названию
- 📅 Фильтрация по диапазону дат
- 🌙 Поддержка light/dark theme
- 🔄 Синхронизация изменений с backend
- ⚡ Debounce обновления при перемещении задач
